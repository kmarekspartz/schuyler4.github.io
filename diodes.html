Semiconductors

So we have finally covered passive components. Now we can get into some real electronics. For real electronics we need semiconducotrs. A semiconductor material is basically a materical that changes its conductivity (resistance) based on another quantity, voltage/temperature. A semiconductor component is an electrical component that exploits this.

IV Curve

This is a usefull tool for understanding semiconductor behavior. Basically we sweep a voltage across a component while also measureing the current throught it. Then the voltage is plotted on the X axis and the current is plotted on the Y axis. The IV curve for a resistor is linear:

Diode IV Curve

Here is an IV curve for a 1N4148 diode that I took with my crappy homebrew curve tracer:

As can be observed, the no current flows through diode until the voltage is ~0.7V in the positive axis. A diode is in some sense a one way conductor. Current will only flow in one direction throught the diode. Additionally, a diode also has a forward voltage; if the voltage across the diode does not exceed the forward voltage then current will not flow even in the diodes forward direction. As can be obvserved from the curve trace, this forward voltage is only an approximation, and really the current through a diode is a steep exponential function of the voltage across it. This steep exponential function can be approximated with a curve trace.
